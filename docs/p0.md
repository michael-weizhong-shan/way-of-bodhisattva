# 入菩萨行论

<div style={{ textAlign: 'center', fontSize: '0.8rem', color: '#666' }}>
  寂天菩萨
</div>

import PoemWrapper from '@site/src/components/poem-wrapper';

<PoemWrapper>

<p data-n="1">善逝法身佛子伴，及诸应敬我悉礼。</p>
<p data-n="2">今当依教略宣说，趋入佛子律仪法。</p>
<p data-n="3">此论未宣昔所无，诗韵吾亦不善巧，</p>
<p data-n="4">是故未敢言利他，为修自心撰此论。</p>
<p data-n="5">循此修习善法故，吾信亦得暂增长，</p>
<p data-n="6">善缘等同诸学人，若得见此容获益。</p>

</PoemWrapper>
